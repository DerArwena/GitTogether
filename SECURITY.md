# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in GitTogether, please report it privately.

Do **not** report security issues through the public issue tracker. Instead, send an email to the project maintainers.

You can expect:
- A confirmation of receipt within 48 hours
- A status update within 5 business days
- A fix or mitigation within 30 days, depending on severity

## Scope

Security issues include:
- Authentication bypass
- Authorization / privilege escalation
- SQL injection
- Cross-site scripting (XSS)
- Exposure of sensitive data
